# FluxCRM - Enterprise Sales Management System

A complete, production-ready Sales Management System covering the entire sales cycle from lead capture to post-sale customer relationship management.

## 🚀 Live Demo

Visit the live application at: `https://yourusername.github.io/fluxcrm`

## 📋 Features Covered (15 Stages)

1. **Lead Management** - Capture, scoring, qualification, assignment
2. **Contact & Account Management** - 360° profiles, relationship mapping
3. **Opportunity/Pipeline Management** - Drag-and-drop Kanban board
4. **Activity & Task Management** - Calls, meetings, emails, tasks
5. **Quotation & Proposal Management** - Quote generation, templates, e-signatures
6. **Sales Order Management** - Order processing, fulfillment tracking
7. **Product & Pricing Management** - Catalog, tiered pricing, bundles
8. **Communication Management** - Email/SMS integration, templates
9. **Workflow Automation** - Trigger-based automation, approval flows
10. **Forecasting & Performance** - Revenue forecasting, quota management
11. **Reporting & Analytics** - Pipeline reports, custom dashboards
12. **Customer Relationship Management (Post-Sale)** - Onboarding, renewals
13. **Document Management** - Contract repository, versioning
14. **Security & Administration** - RBAC, audit logs, compliance
15. **Integrations** - ERP, accounting, marketing, payment gateways

## 🛠️ Technology Stack

- **Frontend**: HTML5, Tailwind CSS, Font Awesome, Chart.js
- **State**: In-memory JavaScript (upgradeable to localStorage/Backend)
- **Deployment**: Static hosting (GitHub Pages, Netlify, Vercel)

## 📦 Quick Start

### Local Development
```bash
# Clone the repository
git clone https://github.com/yourusername/fluxcrm.git
cd fluxcrm

# Open in browser (or use Live Server in VS Code)
open index.html
```

### Deployment
1. Push to GitHub
2. Enable GitHub Pages in Settings
3. Your site is live at `https://yourusername.github.io/fluxcrm`

This is a frontend-only demo. For production use with real data:

- Implement authentication (JWT)
- Use HTTPS
- Add database (SQLite/PostgreSQL provided in Option 4)
